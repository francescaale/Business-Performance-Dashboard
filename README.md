# Business Sales Performance Analytics Dashboard

## Project Overview

The objective was to analyze an online retail sales dataset to identify revenue trends, top-performing products, customer purchasing patterns, and country-level sales performance. An interactive Power BI dashboard was developed to present key business insights and actionable recommendations.

---

## Objectives

- Clean and prepare raw sales data
- Analyze business performance using key KPIs
- Identify revenue trends over time
- Discover top-selling products
- Evaluate sales by country
- Identify high-value customers
- Present findings through an interactive dashboard

---

## Tools Used

- Microsoft Power BI
- Power Query
- DAX (Data Analysis Expressions)
- Microsoft Excel/CSV
- GitHub

---

## Dataset

**Dataset:** Online Retail Dataset

The dataset contains historical online retail transactions, including:

- Invoice Number
- Product Description
- Quantity Sold
- Unit Price
- Invoice Date
- Customer ID
- Country

---

## Key Performance Indicators (KPIs)

- Total Revenue
- Total Orders
- Total Quantity Sold
- Average Order Value
- Total Customers
